<H1>Unit Conversion CLI Tool</H1>

<p>This is a command-line interface (CLI) application designed to perform various mathematical unit conversions. It provides a user-friendly menu system to navigate between different conversion types, including length, temperature, weight, and speed.</p>

<H2>Key Features</H2>

<ul>
    <li><B>Interactive Menu:</B> A clear navigation system allowing users to select from five different conversion types or exit the program.</li>
    <li><B>Robust Input Validation:</B> The tool uses try-except blocks to handle non-numeric inputs gracefully, ensuring the program doesn't crash if a user enters invalid data.</li>
    <li><B>Modular Design:</B> The program separates concerns by utilizing helper modules for screen clearing and specific conversion formulas.</li>
    <li><B>Continuous Workflow:</B> After a calculation, users can choose to perform another conversion of the same type, return to the main menu, or exit the application.</li>
</ul>

<H2>Supported Conversions</H2>

<ol>
    <li><B>Length:</B> Inches to Millimeters</li>
    <li><B>Temperature:</B> Fahrenheit to Celsius</li>
    <li><B>Weight:</B> Pounds to Kilograms</li>
    <li><B>Speed:</B> Miles per hour (mph) to Kilometers per hour (km/h)</li>
    <li><B>Aviation/Marine Speed:</B> Knots to Miles per hour (mph)</li>
</ol>
